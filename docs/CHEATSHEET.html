Key Rules (SYSTEM_PROMPT)

IntentClassify before acting
EvidenceProof before complete
3 FailuresSTOP ‚Üí REVERT ‚Üí ESCALATE

7-Section Delegation

1. TASK: Atomic goal
2. EXPECTED OUTCOME: Success criteria
3. SKILLS: Required expertise
4. TOOLS: Allowed commands
5. MUST DO: Requirements
6. MUST NOT: Forbidden
7. CONTEXT: Files, patterns

üåÄ Flywheel Commands

Setup

flywheel doctor              # Verify tools
flywheel init                 # Bootstrap repo
flywheel checkout feat/x main

Planning (--max = until approved)

flywheel plan --rounds 2
flywheel refine --max 5        # Until approved
flywheel refine --max 5 --bg   # Background
flywheel beads --min-beads 50

Execution

flywheel startwork --cc 3 --cod 2 --gmi 1

Multiple Projects

cd /data/projects/proj1 && flywheel refine --bg
cd /data/projects/proj2 && flywheel refine --bg
