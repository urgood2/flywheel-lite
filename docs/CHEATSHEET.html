üåÄ Flywheel-Lite Commands

Setup & Check

flywheel doctor              # Verify all tools
flywheel init                 # Bootstrap repo
flywheel checkout feat/x main # Branch

Planning Phase

flywheel plan --rounds 2     # Interview
flywheel refine --iters 3    # Codex refine
flywheel beads --min-beads 50

Execution Phase

flywheel startwork --cc 3 --cod 2 --gmi 1

Config: ~/.config/flywheel/config.env

FW_CC=3           # Default Claude agents
FW_COD=2          # Default Codex agents
FW_GMI=1          # Default Gemini agents
FW_PLAN_ITERS=3   # Refinement rounds

üîß Troubleshooting

Agent Stuck

ntm interrupt SESSION
ntm health SESSION

Kill & Respawn

echo y | ntm kill SESSION
ntm spawn SESSION --cc=2

Beads Not Found

cd /data/projects/YourRepo
flywheel init  # or: br init

System Health

flywheel doctor
ntm doctor
br doctor

Session Name Mismatch

# Session must match dir name!
cd /data/projects/myrepo
ntm spawn myrepo --cc=2

ntm send Fails

# Wait 3s after spawn, or use:
ntm send SESSION --pane=2 "msg"
# Instead of --cc=
